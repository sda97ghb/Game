Подробное описание

template<typename T>
class ObjectCounter< T >

Подсчитывает количество объектов указанного типа и возвращает их список.

Использование:

  1. В каждом конструкторе класса, объекты которого надо подсчитывать:
    ObjectCounter<your class name>::addObject(this);
  2. В дестркторе класса, объекты которого надо подсчитывать:
    ObjectCounter<your class name>::removeObject(this);
  3. В любом месте, где вам это нужно:
    • Узнать количество объектов:
      ObjectCounter<your class name>::count()
    • Получитьсписок всех объектов:
      ObjectCounter<your class name>::objects()
      // => std::list<your class name pointer>
Заметки
Этот класс не может подсчитывать объекты, которые не используют addObject и removeObject в их конструкторах и деструкторе. Например вы не можете при помощи этого класса узнать список всех std::string или int использованных в вашей программе.
Объявления и описания членов класса находятся в файле:
  • Arena/ObjectCounter.h
