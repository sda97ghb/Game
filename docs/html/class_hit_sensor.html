Подробное описание

Сенсор столкновения с другим объектом. (Не путать с ContactSensor)

Создает для указанного тела новую фигуру (прямоугольник), чувствительную к взаимодейтвию с другим телом и регистрируется в классе SensorsListener. В отличии от ContactSensor срабатывает только в момент столкновения и не умеет проверять существавание контакта в текущий момент.

Заметки
Для работы с этим датчиком унаследуйте свой класс от Entity или производных от него.

Методы

◆ hangOnBody()

void HitSensor::hangOnBody ( b2Body *  body)

Вешает сенсор на указанное тело.

Запоминает указанное телов в _body, создает для него фигуру-сенсор, регистрирует сенсор в SensorsListener.

Предупреждения
Должен быть вызвано один раз для одного тела.
При повторном вызове для разных тел датчик не будет снят с предыдущего тела.
Заметки
Указанное тело не обязательно должно принадлежать существу, указанному ранее.

◆ setEntity()

void HitSensor::setEntity ( Entity *  entity)

Запоминает существо для последующего обратного вызова метода удара.

Заметки
Запоминает существо, но не владеет им.
Для нормальной работы сенсора в классе указанного существа нужно определить метод void hit(int sensorType, float speed).
Вы можете повешать сенсор на тело, не принадлежащее указанному существу.

◆ setType()

void HitSensor::setType ( int  type)

Устанавливает тип сенсора.

Нужно, чтобы различать сенсоры повешанные на одно тело.

Аргументы
[in]typeтип сенсора, например SensorsListener::GROUND_HIT_SENSOR_TYPE или произвольное целое.
Предупреждения
У одного тела не должно быть несколько сенсоров одинакового типа.
Должен быть вызван раньше, чем hangOnBody(b2Body* body).
