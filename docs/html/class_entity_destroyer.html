Подробное описание

Класс уничтожитель для существ. Корректно уничтожает существо.

Появление этого класса связано с несколькими причинами:

  1. существ можно уничтожать только в промежутках, между обновлениями мира, что делает невозможным их уничтожение непосредственно из класса логики
  2. у существ помимо логики есть еще отображение и физическое тело. EntityDestroyer корректно уничтожит их все.
